# 认证服务基准测试

> 作者 张磊

## 数据估算

* 注册用户数 R 3500
* 登录用户数据（日志系统获得约400） N
* 登录用户数平均时长（估算4小时） L
* 考察时间长度（估算8小时） T
* 并发用户数估算 C 

  * 公式一：N*L/T ，约等于 200
  * 公式二：R\*(8%~12%)，约等于 280～420
  * 并发用户峰值 C^1=(C+3√C)，约等于 243
* TPS=PV\*0.8/(9\*3600)*3
